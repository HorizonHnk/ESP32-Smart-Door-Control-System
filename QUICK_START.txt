⚡ QUICK START GUIDE - ESP32 Voice Control
================================================================================

🎯 GET VOICE CONTROL WORKING IN 5 MINUTES!
===========================================

STEP 1: Upload Code to ESP32
-----------------------------
1. Open Arduino IDE
2. Open: web_integrated_door_control.ino
3. Select: Board = "ESP32 Dev Module"
4. Select: Port = Your ESP32 COM port
5. Click Upload ⬆️
6. Wait for "Done uploading"

STEP 2: Get ESP32 IP Address
-----------------------------
1. Open Serial Monitor (Ctrl+Shift+M)
2. Set baud rate to 115200
3. Press ESP32 reset button
4. Look for: "IP Address: 192.168.x.x"
5. WRITE DOWN THIS IP! Example: 192.168.1.100

STEP 3: Start ngrok
--------------------
1. Open Command Prompt / PowerShell
2. Type: ngrok http YOUR_ESP32_IP:80
   Example: ngrok http 192.168.1.100:80
3. Press Enter
4. Look for line: "Forwarding https://xxxxx.ngrok.io -> http://..."
5. COPY THE HTTPS URL: https://xxxxx.ngrok.io

STEP 4: Access Dashboard
-------------------------
1. Open Chrome or Edge browser
2. Paste your ngrok HTTPS URL
3. Dashboard loads ✅

STEP 5: Use Voice Control
--------------------------
1. Click "🤖 AI Chat" tab
2. Click yellow 🎤 button
3. Allow microphone when asked
4. Button turns red 🔴
5. Say: "Open the door"
6. Watch it work! 🎉

🆘 NEED HELP?
=============

1. Microphone not working?
   → Make sure you're using HTTPS (ngrok URL)
   → Use Chrome or Edge browser
   → Check microphone permissions

2. Voice not recognized?
   → Speak clearly and slowly
   → Use exact commands from list
   → Reduce background noise

3. ngrok URL not working?
   → Check ESP32 IP is correct
   → Make sure ESP32 is powered on
   → Verify ngrok is still running

4. ESP32 not responding?
   → Check Serial Monitor for errors
   → Verify WiFi connection (ESP32SEG)
   → Try resetting ESP32
