STEP 2: Get ESP32 IP Address
-----------------------------
1. Open Serial Monitor (Ctrl+Shift+M)
2. Set baud rate to 115200
3. Press ESP32 reset button
4. Look for: "IP Address: 192.168.x.x"
5. WRITE DOWN THIS IP! Example: 192.168.1.100

STEP 3: Start ngrok
--------------------
1. Open Command Prompt / PowerShell
2. Type: ngrok http YOUR_ESP32_IP:80
   Example: ngrok http 192.168.1.100:80
3. Press Enter
4. Look for line: "Forwarding https://xxxxx.ngrok.io -> http://..."
5. COPY THE HTTPS URL: https://xxxxx.ngrok.io

STEP 4: Access Dashboard
